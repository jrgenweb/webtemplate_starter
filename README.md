# Basic packages and file structure for creating static websites.

## Included packages:
- PostCSS
- CSSNano
- Sass
- Dart-Sass
- Gulp
- Gulp-Babel
- Gulp-PostCSS
- Gulp-Sass
- Gulp-Terser
- Babel/Core
- Babel/Preset-env
- AutoPrefixer
- Browser-Sync

## Usage

After cloning, type the following command line commands

- ``` npm install ```
- ``` gulp ```
